# Private Evidence Policy

This repository is public. Do not commit private verification evidence.

Never add:

- Witness names or witness emails unless already intentionally public.
- Addresses, phone numbers, or birth dates.
- Completed registration forms.
- Photos or scans submitted for verification.
- Private correspondence.
- Internal review notes that identify private witnesses.

Use a separate private storage location for evidence, such as a private drive,
private database, or private repository with tightly controlled access.

Public data in this repository should be limited to the ranking information that
is meant to appear on the website.
